## Lo que aprendí

Practiqué cómo usar `grid-column: span 2` para que una sección ocupe todo el ancho del grid en desktop sin necesidad de reestructurar el HTML. También usé `clamp()` para que la tarjeta tenga un ancho fluido con límites mínimo y máximo sin media queries adicionales.
